# Clean the Ocean Game

A web-based game where you control a fish to clean up ocean pollution.

## How to Play

- **Desktop:** Use WASD or arrow keys to move. Optionally move towards mouse cursor.
- **Mobile:** Tap or drag on the screen to move the fish.

Collect trash to earn points and restore ocean health. Avoid pollution clouds.

## Features

- Responsive design that works on desktop and mobile devices
- Touch controls for mobile gameplay
- Multiple difficulty levels

## Running the Game

Open `index.html` in a web browser to start playing.
